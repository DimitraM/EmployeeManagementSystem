# ReactJS + Spring Boot CRUD Full-Stack Application

This is a full-stack CRUD (Create / Read / Update / Delete) application built using **React** for the frontend and **Spring Boot** (with Spring Data JPA + a relational database) for the backend.  
It follows the tutorial series from the accompanying video playlist.
